Démarrage rapide

Prérequis

  • Navigateur Chromium
  • Compte Google (optionnel, requis pour Premium)
  • Connexion internet pour la traduction

Installation

  • Ouvrez le Chrome Web Store et cherchez "Reborn Vision"
  • Cliquez sur "Ajouter à Chrome"
  • Épinglez l'icône dans la barre d'outils
  • Cliquez sur l'icône pour ouvrir le panneau Vision
